三、属性选择器

1. E[arrt] 匹配所有具有attr属性的E元素

2. E[arrt=val] 匹配所有具有attr=val属性的E元素

3. E[arrt~=val] 匹配所有att属性具有多个空格分隔的值、其中一个值等于“val”的E元素

4. E[arrt^=val] 匹配属性值以指定值开头的每个元素

4. E[arrt^=val] 匹配属性值以指定值开头的每个元素

5. E[arrt$=val] 匹配属性值以指定值结尾的每个元素

5. E[arrt$=val] 匹配属性值以指定值结尾的每个元素

6. E[arrt*=val] 匹配属性值中包含指定值的每个元素

6. E[arrt*=val] 匹配属性值中包含指定值的每个元素
